## DESCRIPCIÓN DEL JUEGO:
Typing Leaf es un juego donde el usuario deberá usar el teclado para poder interactuar con la canasta, aparecerán hojas de árbol con letras incluidas en ellas, habrá un objetivo en concreto el cual será armar una palabra aleatoria que aparecerá en pantalla con las letras correspondientes que vayan saliendo, cada vez que se recoge una hoja de árbol, debe coincidir esa letra con la palabra aleatoria, si se recoge una letra equivocada, se deja caer una que ya fue elegida anteriormente o se deja caer al suelo, se perderán puntos, de lo contrario, se ganarán.
